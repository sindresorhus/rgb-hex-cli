# rgb-hex-cli

> Convert RGB color to HEX


## Install

```
$ npm install --global rgb-hex-cli
```


## Usage

```
$ rgb-hex --help

  Examples
    $ rgb-hex 255 154 253
    ff9afd
    $ rgb-hex 'rgb(40, 42, 54)'
    282a36
    $ rgb-hex 'rgba(40, 42, 54, 75%)'
    282a36c0
```
